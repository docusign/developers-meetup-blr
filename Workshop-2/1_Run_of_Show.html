From Static PDFs to Agentic Workflows

Run of Show · Cross-functional business stakeholders · Developer & implementation-partner audience

Who this event is for
Engineering and product practitioners, architects, and IT platform leaders focused on building agreement-centric features and modernizing enterprise workflows (sales, finance, and operations) across their core business systems.
Why attend
Build agentic agreement workflows with Docusign IAM. Turn signed vendor contracts into an AI-ready pipeline using the Docusign CLI and Agreement CLI to model MSAs, SOWs, and NDAs as code and bulk-ingest legacy repositories. Let Iris AI extract key procurement terms, then use the MCP Server and Copilot Studio to ask natural-language questions like “which vendor agreements auto-renew in the next 90 days?”, and close the loop with an Agent Studio AI agent that surfaces renewals, off-standard terms, and obligations where your procurement team works.
End-to-end flow
CLI login & upload → extraction in Navigator / Agreement Manager → MCP via Copilot Studio → Agent Studio demo
Timing note: the durations below reflect the updated cross-functional / developer flow and total roughly 65–75 minutes. The MCP section is the heart of the session (30 min); the CLI hands-on is deliberately tight (15 min) and Agent Studio is a quick demo (10 min).
