# Workflow Builder

This module covers building automated agreement workflows using Docusign's no-code/low-code **Workflow Builder** (Formerly Maestro)

---

## Overview

**Workflow Builder** lets you design multi-step agreement automation using a visual canvas — no coding required. Use it to:

- Route agreements based on field values or conditions
- Send reminder and follow-up notifications automatically
- Branch workflows based on recipient actions (signed, declined, etc.)
- Integrate with external systems via webhooks

---

## Lab Steps

### Step 1 Turn ON Agreements Desk
1.  Log in to your Docusign Developer Account
2.  Go to **Agreements > Requests**
3.  Select **Turn ON Agreement Desk**

### Step 2 Turn ON Agreements Mange Access
1.  Log in to your Docusign Developer Account
2.  Go to **Admin > Users**
3.  Select **Actions** on your user id row.
4.  Select **Manage Agreement Desk Access**
5.  Toggle ON both the seetings:
   * **Manage all requests in the account**
   * **Configure Agreement Desk settings**
6. Select **Save**.

### Step 3  Access Workflow Builder

1. Download the workflow files available in  [`05-Workflow-Builder/exports/`](./exports/)
2. Log in to your Docusign Developer Account
3. Go to **Agreements > Workflows**
4. Select **Create Workflow**
5. Select **Import Workflow**.
7. Import both the workflow builder files.
  * ```workflow-1-name```
  * ```workflow-1-name```
