# Workflow Builder

This module covers building automated agreement workflows using Docusign's no-code/low-code **Workflow Builder** (Formerly Maestro)

---

## Overview

**Workflow Builder** lets you design multi-step agreement automation using a visual canvas — no coding required. Use it to:

- Route agreements based on field values or conditions
- Send reminder and follow-up notifications automatically
- Branch workflows based on recipient actions (signed, declined, etc.)
- Integrate with external systems via webhooks

---

## Lab Steps

### Step 1 — Access Workflow Builder

1. Log in to your Docusign Developer Account
2. Go to **Agreements > Workflows**
3. Select **Create Workflow**
4. Select **Import Workflow**.
5. Upload bot the workflow files available in  [`05-Workflow-Builder/exports/`](./05-Workflow-Builder/exports/)
6. Import the file from exports folder.
  * Workflow -1 and workflow-2

Follow the demo for the next steps of configuring workflows in Docusign Workflow Builder.
