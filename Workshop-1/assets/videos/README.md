# Videos

This folder is for demo and walkthrough video files used in the workshop.

## Guidelines

- Keep individual video files under **100MB** (GitHub's file size limit)
- For larger videos, upload to a video hosting service (YouTube, Loom, Vimeo) and link from the relevant `.md` file instead
- Use the same naming convention as images: `<module-prefix>-<description>.mp4`

## Embedding Videos in Markdown

For hosted videos, link them as follows:

```markdown
[![Video Title](../assets/images/video-thumbnail.png)](https://www.youtube.com/watch?v=VIDEO_ID)
```
