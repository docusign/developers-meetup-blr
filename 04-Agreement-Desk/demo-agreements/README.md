# Demo Agreements

This folder contains sample agreement files used in the **Agreement Desk** and **AI-AR** lab exercises.

## Files

| File | Type | Use In |
|------|------|--------|
| `vendor-nda-signed.pdf` | Vendor NDA (completed/signed) | Agreement Desk import lab |
| `msa-template.pdf` | Master Service Agreement | AI-AR extraction lab |
| `sow-example.pdf` | Statement of Work | Agreement Desk custom fields lab |
| `renewal-ready-contract.pdf` | Contract near expiration | Renewal reminder lab |
| `high-risk-sample.pdf` | Contract with non-standard clauses | AI-AR risk scoring lab |

## Usage Instructions

### For Agreement Desk Labs
1. Upload files via **Agreement Desk > + New Agreement > Upload a file**
2. Fill in metadata fields as instructed in [Agreement-Desk.md](../Agreement-Desk.md)

### For AI-AR Labs
1. Upload or open the agreement in Agreement Desk
2. Click **AI Review** in the right-side panel
3. Compare AI-extracted fields against the actual document

## Notes

- All agreements in this folder are **fictional** and for training purposes only
- Party names, emails, and amounts are placeholder data
- Do not use these files for any real business purpose

## Adding Your Own Demo Files

When adding new demo agreements to this folder:
- Ensure all personal data is anonymized or fictional
- Use descriptive filenames (lowercase, hyphens)
- Update the table above with a brief description
