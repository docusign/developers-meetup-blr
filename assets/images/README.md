# Images

This folder contains screenshots and diagrams used throughout the workshop guides.

## Naming Convention

Use the module prefix when naming files:

| Prefix | Module |
|--------|--------|
| `01-` | Overview |
| `02-` | Docusign Salesforce / Agentforce |
| `03-` | Workflow Builder |
| `04-` | Agreement Desk |
| `05-` | AI-AR |

**Examples:**
- `02-architecture-diagram.png`
- `02-named-credential.png`
- `03-workflow-builder-canvas.png`
- `04-agreement-desk-dashboard.png`
- `05-ai-review-panel.png`

## Format

- Preferred format: **PNG** for screenshots, **SVG** for diagrams
- Max width: **1200px** (resize before uploading to keep repo size manageable)
- Use descriptive filenames in lowercase with hyphens (no spaces)
